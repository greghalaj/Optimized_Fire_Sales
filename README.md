# Optimized_Fire_Sales
Based on a model of Feinstein and Halaj (JEDC, 2023), this code allows to run simulations of how price-mediated contagion arises when banks respond to funding shocks by selling securities to raise cash. They choose securities by solving an optimal portfolio program.
